
# 第3部 サンプルプロンプト一覧

ここでは、研修内で使用する**分析用のデータ**と**サンプルプロンプト**をまとめています。
各自、必要なタイミングで参照をお願いします。

---

### 役割

```
あなたは行政コンサルタントです。
```

```
あなたは長岡市に詳しい地域経済アナリストです。
```

### 形式

```
この結果からわかることを箇条書きで解説してください。
```

```
分析結果を表形式にまとめて出力してください。
```

---

### Copilot演習

### ステップ➀

```
各パラメータの平均値、中央値、分散、標準偏差を算出してください。
```

### ステップ➁

```
「高齢化率（65歳以上）」と「人口増減率」の2パラメータを使って
散布図を作成してください。
```

```
散布図に回帰線を引き、決定係数も算出してグラフ内に表示してください。
```

### ステップ③

```
これまでの結果を見て、長岡市の高齢化にどのような要因があるのかを分析してください。
さらに、どのような課題があるのかを説明してください。
また、課題に対する解決方法も提案してください。
```

---

### テキストマイニング

### アンケート分析➀：クロス集計

```
年代別の満足度を集計して、 表形式で出力してください。
```

```
また、満足度が低い年代について、
その理由を自由記述から推測して考察してください。
```

### アンケート分析②：ワードクラウド（頻出語）

```
自由記述欄から、頻出する単語トップ20を抽出し、
その出現回数とセットで表にまとめてください。
```

### アンケート分析③：共起ネットワーク（文脈）

```
「〇〇」という単語と一緒に使われている言葉（共起語）は何ですか？
```

```
それらが使われている具体的な文脈を3つ挙げて解説してください。
```

### アンケート分析④：感情分析

```
各回答の内容を読み取り、「ポジティブ」「ネガティブ」「中立」の3段階で
判定してください。
```

```
それぞれの割合を集計して、円グラフ用のデータとして
出力してください。
```
