## Copilot演習

### ステップ➀

```
各パラメータの平均値、中央値、分散、標準偏差を算出してください。
```

### ステップ➁

```
各パラメータの相関係数を算出し、相関行列として出力してください。
また、ヒートマップで視覚化してください。
```

```
異なる3つの切り口で、新潟県の「地域課題」や「自治体ごとの特徴」をあぶり出すのに
適したパラメータを選定し、散布図を作成してください。
```

```
散布図に回帰線を引き、決定係数も算出してグラフ内に表示してください。
```

### ステップ③

```
これまでの結果を見て、地域活性化においてどんな課題がありますか？
逆に、どんな解決方法がありますか？
```

## テキストマイニング

### アンケート分析➀：クロス集計

```
年代別の満足度を集計して、 表形式で出力してください。
```

```
また、満足度が低い年代について、
その理由を自由記述から推測して考察してください。
```

### アンケート分析②：ワードクラウド（頻出語）

```
自由記述欄から、頻出する単語トップ20を抽出し、
その出現回数とセットで表にまとめてください。
```

### アンケート分析③：共起ネットワーク（文脈）

```
「〇〇」という単語と一緒に使われている言葉（共起語）は何ですか？
```

```
それらが使われている具体的な文脈を3つ挙げて解説してください。
```

### アンケート分析④：感情分析

```
各回答の内容を読み取り、「ポジティブ」「ネガティブ」「中立」の3段階で
判定してください。
```

```
それぞれの割合を集計して、円グラフ用のデータとして
出力してください。
```
